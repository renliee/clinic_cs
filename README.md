# Clinic CS AI (RAG)

AI customer service chatbot for clinic booking system using Retrieval-Augmented Generation (RAG).

## Features

- FAQ answering using RAG
- Booking conversation flow
- Session memory per user
- Booking stored in database

## Tech Stack

- Python
- LangChain
- Ollama LLM (llm used: qwen2.5:14b)
- Chroma Vector DB
